## 三、核心模块说明

### 3.1 LLM客户端 (backend/utils/llm_client.py)

**功能**：封装阿里云大模型API调用

**使用示例**：
```python
from backend.utils.llm_client import llm_client

response = llm_client.chat_with_system_prompt(
    system_prompt="你是一个需求分析师",
    user_prompt="请分析以下需求：..."
)
```

### 3.2 文档解析器 (backend/utils/docx_parser.py)

**功能**：解析.docx文档，提取需求内容

**使用示例**：
```python
from backend.utils.docx_parser import docx_parser

result = docx_parser.parse("需求文档.docx")
# 返回: {"requirement_name": "...", "requirement_content": "..."}
```

### 3.3 Agent编排器 (backend/agent/agent_orchestrator.py)

**功能**：编排各Agent协同工作

**工作流程**：
1. 系统识别（支持知识库检索）
2. 功能点拆分
3. 工作量估算（Delphi法）
4. 报告生成

### 3.4 知识库服务 (backend/service/knowledge_service.py)

**功能**：管理Milvus向量数据库

**主要方法**：
- `upload_document()` - 上传文档到知识库
- `search()` - 检索相似系统案例
- `delete_document()` - 删除文档

## 四、新增Service层

### 4.1 知识库服务

**位置**: `backend/service/knowledge_service.py`

**功能**：
- 管理Milvus Collection
- 文档向量化和存储
- 相似度检索
- 注入Agent提示词

**使用示例**：
```python
from backend.service.knowledge_service import get_knowledge_service

service = get_knowledge_service()

# 上传文档
service.upload_document("系统文档.docx", doc_id)

# 检索
results = service.search("支付中台系统", top_k=5)
```

### 4.2 Embedding服务

**位置**: `backend/service/embedding_service.py`

**功能**：
- 文本向量化
- 批量处理
- 调用阿里云Embedding API

**向量模型**：
- 模型：text-embedding-v2
- 维度：1024
- 相似度：余弦相似度

### 5.2 添加新的API接口

1. 在 `backend/api/` 下创建新的路由文件
2. 定义FastAPI路由器
3. 在 `backend/app.py` 中注册路由

**示例**：

```python
# backend/api/custom_routes.py
from fastapi import APIRouter

router = APIRouter(prefix="/api/v1/custom", tags=["自定义功能"])

@router.get("/hello")
async def hello():
    return {"message": "Hello"}

# backend/app.py
from backend.api.custom_routes import router as custom_router
app.include_router(custom_router)
```

## 十、常见问题

### Q1: 如何切换LLM模型？

修改 `.env` 文件：
```bash
LLM_MODEL=qwen-max  # 或 qwen-plus, qwen-turbo
```

### Q2: 如何启用知识库功能？

1. 部署Milvus（见部署手册）
2. 配置环境变量
3. 上传系统文档到知识库

### Q3: 如何调试Agent？

在Agent中添加日志：
```python
logger.info(f"输入数据: {input_data}")
logger.info(f"LLM响应: {response}")
```
