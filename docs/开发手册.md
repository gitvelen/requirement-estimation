## 三、核心模块说明

### 3.1 LLM客户端 (utils/llm_client.py)

**功能**：封装阿里云大模型API调用

**核心方法**：
```python
class LLMClient:
    def chat(messages, temperature, max_tokens) -> str
    def chat_with_system_prompt(system_prompt, user_prompt) -> str
    def extract_json(text) -> Dict
```

**使用示例**：
```python
from backend.utils.llm_client import llm_client

response = llm_client.chat_with_system_prompt(
    system_prompt="你是一个需求分析师",
    user_prompt="请分析以下需求：..."
)
```

### 3.2 文档解析器 (utils/docx_parser.py)

**功能**：解析.docx文档，提取需求内容

**核心方法**：
```python
class DocxParser:
    def parse(file_path: str) -> Dict[str, str]
    def validate_file(file_path: str) -> bool
```

**使用示例**：
```python
from backend.utils.docx_parser import docx_parser

result = docx_parser.parse("需求文档.docx")
print(result["requirement_name"])
print(result["requirement_content"])
```

### 3.3 Agent编排器 (agent/agent_orchestrator.py)

**功能**：编排各Agent协同工作

**核心方法**：
```python
class AgentOrchestrator:
    def process_requirement(task_id, requirement_data) -> str
    def process_with_retry(task_id, requirement_data, max_retry) -> str
```

**工作流程**：
1. 系统识别
2. 功能点拆分
3. 工作量估算
4. 报告生成

## 四、添加新Agent

### 4.1 Agent模板

```python
"""
自定义Agent模板
"""
import logging
from typing import Dict, List
from backend.utils.llm_client import llm_client

logger = logging.getLogger(__name__)


class CustomAgent:
    """自定义Agent"""

    def __init__(self):
        """初始化Agent"""
        self.name = "自定义Agent"
        logger.info(f"{self.name}初始化完成")

    def process(self, input_data: Dict) -> Dict:
        """
        处理输入数据

        Args:
            input_data: 输入数据

        Returns:
            Dict: 处理结果
        """
        try:
            logger.info("开始处理...")

            # 1. 构建提示词
            system_prompt = "你是一个..."
            user_prompt = f"请处理以下数据：{input_data}"

            # 2. 调用LLM
            response = llm_client.chat_with_system_prompt(
                system_prompt=system_prompt,
                user_prompt=user_prompt
            )

            # 3. 解析结果
            result = llm_client.extract_json(response)

            logger.info("处理完成")
            return result

        except Exception as e:
            logger.error(f"处理失败: {str(e)}")
            raise


# 全局Agent实例
custom_agent = CustomAgent()
```

### 4.2 集成到编排器

在`agent_orchestrator.py`中添加：

```python
from backend.agent.custom_agent import custom_agent

class AgentOrchestrator:
    def process_requirement(self, task_id, requirement_data):
        # ... 原有代码 ...

        # 调用自定义Agent
        custom_result = custom_agent.process(requirement_data)

        # ...
```

## 五、提示词工程

### 5.1 提示词模板位置

所有提示词模板存放在：`backend/prompts/prompt_templates.py`

### 5.2 提示词优化建议

1. **明确角色定位**：
   - 好的提示词："你是一个有10年经验的需求分析师..."
   - 不好的提示词："你是一个助手..."

2. **提供示例**：
   - 在提示词中包含1-2个完整的示例
   - 示例应覆盖常见场景

3. **输出格式规范**：
   - 明确指定JSON格式
   - 提供JSON Schema示例

4. **约束条件清晰**：
   - 功能点粒度：0.5-5人天
   - 复杂度分级标准
   - 依赖关系描述规范

### 5.3 提示词测试

创建测试脚本验证提示词效果：

### 7.2 异步处理

使用FastAPI的BackgroundTasks处理长时间任务：

```python
from fastapi import BackgroundTasks

@router.post("/requirement/upload")
async def upload_requirement(
    file: UploadFile,
    background_tasks: BackgroundTasks
):
    # 创建任务
    task_id = create_task(file)

    # 后台处理
    background_tasks.add_task(process_task, task_id, file_path)

    return {"task_id": task_id}
```
