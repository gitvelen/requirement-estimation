--------------------------------------------------------------------------------
问题描述
--------------------------------------------------------------------------------
内网服务器导入 ESB 文档时出现 HTTP 400 Bad Request 错误
原因: embedding 批次大小（默认 25）超过内网 API 限制（约 2048 tokens）

--------------------------------------------------------------------------------
解决方案
--------------------------------------------------------------------------------
添加可配置参数 ESB_EMBEDDING_BATCH_SIZE，默认值 10
用户可根据实际环境调整（推荐范围: 5-15）

--------------------------------------------------------------------------------
代码变更
--------------------------------------------------------------------------------
1. backend/config/config.py (第 68-69 行)
   + ESB_EMBEDDING_BATCH_SIZE: int = int(os.getenv("ESB_EMBEDDING_BATCH_SIZE", "10"))

2. backend/service/esb_service.py (第 499 行)
   - embeddings = embedding_service.batch_generate_embeddings(texts)
   + embeddings = embedding_service.batch_generate_embeddings(texts, batch_size=settings.ESB_EMBEDDING_BATCH_SIZE)
